# Overseers WICC Resources

This repository functions as a source-of-truth for the Overseers-managed platform at WICC 2026. This repository contains the following components:

- `spec`: A formal specification of gameplay rules and scoring mechanisms.
- `reference-attacker`: A reference implementation of an Attack-Defense attacker capable of retrieving flag IDs, attacking services, and submitting flags to the platform.

The WICC and Overseers team treats this repository as an accurate description of the platform's intended behavior. Any discrepancies between this repository and the actual platform should be reported to the Overseers team. We reserve the right to update this repository to reflect changes in the platform's behavior, and we will make every effort to communicate such changes to participants in a timely manner.
